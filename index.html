Início

Este é o conteúdo da tela. O menu lateral acima é o componente de navegação — reduza a janela para ver o comportamento mobile.

Componente reutilizável

A estrutura de itens vem de MENU_CONFIG, no final do arquivo. Adicione, remova ou reordene itens — inclusive submenus — editando apenas esse array; o HTML é gerado automaticamente.

Comportamento responsivo

Acima de 900px o menu fica fixo na lateral. Abaixo disso, ele vira um painel deslizante acionado pelo botão “Menu”, com camada escurecida sobre o conteúdo.
